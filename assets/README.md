# 이미지 리소스 요청 목록

아래 파일명으로 이미지를 주시면 현재 코드에 바로 교체 반영할 수 있습니다.

## 필수 리소스

- `brand-logo.svg`
  - 용도: 상단 헤더 로고
  - 권장: 가로형 투명 배경 로고
  - 권장 크기: 240x80 이상

- `hero-facility-visual.svg`
  - 용도: 첫 화면 우측 대표 이미지
  - 추천 내용: 운영 거점 또는 공유시설 대표 사진
  - 권장 크기: 1600x1400 이상

- `operation-support-visual.svg`
  - 용도: 중간 설명 섹션 이미지
  - 추천 내용: 상담 장면, 제조 준비 장면, 공간 내부, 지원 프로세스 관련 사진
  - 권장 크기: 1400x1100 이상

## 선택 리소스

- `brand-mark-placeholder.svg`
  - 현재는 동일 플레이스홀더 3개를 사용 중
  - 필요 시 아래 3개 파일로 분리 가능
  - `brand-mark-1.svg`
  - `brand-mark-2.svg`
  - `brand-mark-3.svg`

- `og-image.jpg`
  - 용도: 공유용 썸네일 이미지
  - 권장 크기: 1200x630

## 전달 방식

- 투명 배경 로고는 `SVG` 또는 `PNG`
- 사진은 `JPG` 또는 `PNG`
- 파일명을 그대로 맞춰주시면 코드 수정 없이 교체 가능
